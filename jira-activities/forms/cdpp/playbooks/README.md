Líder Técnico. Las plantillas se extraen de tickets reales aprobados.

Ver `../APRENDIZAJES.md` para el registro completo de tipos, patrones y
lecciones aprendidas.

## Cómo usar un playbook

1. Identificar el tipo de pase (ver `../references/tipos-cdpp.md`).
2. Abrir el `README.md` del playbook correspondiente.
3. Usar las plantillas como base, reemplazando los placeholders `<...>`.
4. Consultar `../APRENDIZAJES.md` para ver patrones y lecciones del tipo.
