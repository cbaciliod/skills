# Changelog

Todos los cambios notables de este proyecto se documentan en este archivo.

El formato sigue [Keep a Changelog](https://keepachangelog.com/es/1.0.0/) y el versionado usa [Semantic Versioning](https://semver.org/lang/es/).

---

## [1.0.0] - 2026-05-22

### Added

- Skill `git-workflow` para Claude Code que genera commits, Pull Requests y CHANGELOG siguiendo Conventional Commits
- Archivo `references/commit-standard.md` con el estándar de commits, tipos permitidos, reglas de formato y estructura del CHANGELOG
- Archivo `references/pr-guidelines.md` con guía de PRs pequeños, límites prácticos y técnicas de división (Branch by Abstraction, Strangler Fig, Parallel Change, descomposición por capas, refactoring incremental)
- Archivo `references/pr-template.md` con el template de título y cuerpo de PR
- `SKILL.md` con la definición del skill, tabla de tareas/referencias, flujo recomendado y reglas prioritarias
